VERSION 12.12

Building and Configuration

Developed under Visual Studio 2012 on Windows 7. The Unit test projects are VS2012 compatible
  • The TfsUrl should be fully formed. In my case it is https://[domain]/tfs/[namespace?]
  • This expects explicit credentials and does not support single sign on, that I know of

Execution and configuration

Edit the App.config file and build the app. There is no binary download at this time (12/12/12)
  • Tell the system where your TFS connection information
  • Select the hardware device or the simulated device
  • The communication port/speed or ethernet device for a hardware build status light, depending how it is connected
